Download2Go Android ExoPlayer 2.14.* Support Library

Support library to simplify using the Penthera SDK with Exoplayer v2.14.

This library provides support functions to simplify integration with Exoplayer. It provides utility functions that will create a complete player instance to play one or more assets, or each component can be used individually to tailor a bespoke player integration. The library includes replacement classes for the Exoplayer DRM framework which will manage offline keys via the Penthera SDK. It also includes methods to aid in creating MediaItems for the player and the Penthera custom analytics listener which will log playback events containing playback quality statistics.
