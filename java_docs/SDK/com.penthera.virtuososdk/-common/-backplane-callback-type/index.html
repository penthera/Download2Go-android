BackplaneCallbackType

class BackplaneCallbackType
Content copied to clipboard

Identifies the type of request or instruction in the IBackplaneObserver callback method. The BackplaneCallbackType is but one of the parameters; it is accompanied with an Integer result value defined in BackplaneResult that can be used to determine success or failure of communication with the Backplane.
