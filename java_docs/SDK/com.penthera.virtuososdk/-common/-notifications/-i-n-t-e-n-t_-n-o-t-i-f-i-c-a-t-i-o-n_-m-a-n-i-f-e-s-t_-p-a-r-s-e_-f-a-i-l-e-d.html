INTENT_NOTIFICATION_MANIFEST_PARSE_FAILED

val INTENT_NOTIFICATION_MANIFEST_PARSE_FAILED: String = ".NOTIFICATION_MANIFEST_PARSE_FAILED"

Broadcast when a recently queued manifest fails to be fully parsed but the receiving app has been backgrounded and the registered observer is no longer available to notify of the error. This may occur due to an issue with the manifest or it may be due to the parser being destroyed when the device reclaims memory while the app is in the background. These occurrences should be rare.
