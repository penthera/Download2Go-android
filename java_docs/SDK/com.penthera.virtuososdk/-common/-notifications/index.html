Additional info in notifications to explain why Virtuoso stopped downloading One of these values is sent in the notification Intent for virtuoso.intent.action.NOTIFICATION_DOWNLOAD_STOPPED or virtuoso.intent.action.NOTIFICATION_DOWNLOADS_PAUSED, and can be retrieved from the extras with the key EXTRA_NOTIFICATION_DOWNLOAD_STOP_REASON.
