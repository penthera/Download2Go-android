EngineBlockedReason

class EngineBlockedReason

All the reasons that the download engine would report status as blocked. Provided as flags in notifications when the status is blocked and via the Virtuoso interface. The cause could be more than one reason, but only the first reason which triggered the engine to enter the blocked state will be reported in notifications.
