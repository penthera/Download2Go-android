VirtuosoFCMMessageHandler

open class VirtuosoFCMMessageHandler

This class handles messages delivered from Firebase messaging for the SDK. It will accept any Firebase RemoteMessage, process the message, and indicate if the message was intended for the SDK. If implementing a custom FCM message service in your application then please add this handler class and call the respective methods for new tokens and for received messages. The SDK requires access to the tokens in order to update the cloud server and ensure targeted push messages can be delivered to devices. This message handler is used within the default Firebase service provided, so if push messaging is not being used elsewhere in the application then that class can be registered directly in the Android manifest.
