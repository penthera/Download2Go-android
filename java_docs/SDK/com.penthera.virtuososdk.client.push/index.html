Package com.penthera.virtuososdk.client.push

Classes for support Firebase or Amazon push messaging services with the SDK.

Push messaging enables the server to pass device specific messages to improve response times when settings are updated or the device is required to enact business rules. Actions such as asset deletion or remote wipe of the device would otherwise only happen upon the next scheduled server sync.
