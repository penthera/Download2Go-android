Class for communicating with Firebase Messaging. This class does not always need to be overridden. Only implement a subclass if you need to handle messages other than those delivered by the SDK and wish to do it within a derived class. Alternately implement a custom messaging service and pass messages to [VirtuosoFCMMessageHandler]. That class can be sent a message and will indicate if the message was intended for the SDK and has been processed.
