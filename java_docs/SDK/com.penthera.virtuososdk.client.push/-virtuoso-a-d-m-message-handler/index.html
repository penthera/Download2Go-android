VirtuosoADMMessageHandler

open class VirtuosoADMMessageHandler : ADMMessageHandlerImpl

This class encapsulates ADM message handling for the Virtuoso SDK in a way that does not require extension of any other Virtuoso SDK classes. This is useful for implementations that already have their own ADM message handling and need to process ADM messages outside the Virtuoso SDK.
