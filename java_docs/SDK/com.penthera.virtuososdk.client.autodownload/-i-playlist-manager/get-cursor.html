abstract fun getCursor(): Cursor

Queries the Uri, returning a Cursor over the result set. The Cursor will default to retrieving all fields and all rows from the Content provider. The sorting order on the Cursor is determined by the internal implementation. Cursor related methods should not be used on the main thread as they rely upon blocking actions in the database.

Return

A Cursor object, which is positioned before the first entry, or null.
