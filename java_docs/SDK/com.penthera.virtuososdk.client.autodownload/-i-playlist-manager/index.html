interface IPlaylistManager

IPlaylistManager is used to manage playlists in the autodownload feature. Each IPlaylist contains a collection of IPlaylistItems. Each item represents a single IAsset that is part of the playlist. The playlist manager is used to manage creation, updates to, and deletion of Playlists.
