abstract fun didProcessAssetForPlaylist(asset: IIdentifier, playlistName: String): IIdentifier

Indicates that a new asset has been created. This method will be called as soon as new asset items have been created and prior to being added to the download queue. The provider may further modify the asset object, such as updating the metadata or adding file-unique headers. It is expected that the implementation of this method will call the update method prior to returning and that any changes made within this method will be made synchronously.

Return

The updated asset which has been updated and stored if necessary

The created asset
