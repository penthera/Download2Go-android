interface IPlaylistAssetProvider
Content copied to clipboard

This interface provides a configurable mechanism to populate asset details when the Virtuoso engine creates new assets automatically using the auto download feature. The interface must be implemented to provide this function if auto downloads are being used in an application. It will need to access appropriate catalog data to populate the necessary asset fields to enable a download to be initiated. The class implementing IPlaylistAssetProvider will be called from a androidx.work.WorkManager based androidx.work.Worker, and may be called when the app is in the background. It will only be called when the SDK determines that the device is online. The IPlaylistAssetProvider is provided to the engine through the IBackgroundProcessingManager interface, which itself is registered via the Android manifest.
