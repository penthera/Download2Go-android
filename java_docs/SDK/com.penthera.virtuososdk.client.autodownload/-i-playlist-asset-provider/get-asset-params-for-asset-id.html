getAssetParamsForAssetId

abstract fun getAssetParamsForAssetId(assetId: String): VirtuosoPlaylistAssetItem

Implement this method to provide details of an asset, given the asset id and playlist to which it belongs. The caller is expected to do a synchronous lookup of the catalog data, based on the caller-defined external asset ID, and return AssetParams which will be used to create the Asset and add it to the download queue. If the asset details are temporarily not possible to populate then return null and the engine will make the request again after a time interval and when the device is online. If the asset is no longer available in the catalog then throw an AssetNoLongerAvailableException and the engine will remove the asset from the queue and make no further requests for the details. Use com.penthera.virtuososdk.client.builders.HLSAssetBuilder to assemble the parameters for an HLS asset or use com.penthera.virtuososdk.client.builders.MPDAssetBuilder to assemble the parameters for an MPEG DASH asset, or use com.penthera.virtuososdk.client.builders.FileAssetBuilder to assemble the parameters for an single file asset.

Return

an AssetParams derived instance, which can be generated using an AssetBuilder

Parameters

assetId

The asset id for which to provide the params

Throws

AssetNoLongerAvailableException

if the required asset is no longer available in the catalog
