IPlaylist

interface IPlaylist

The playlist represents a sequenced set of Assets that are to be consecutively downloaded. Binge watching a TV series provides an appropriate example for how Playlists are intended to be used. When a TV show in the series is watched and deleted, the Playlist will automatically start downloading the next TV show in the series.
