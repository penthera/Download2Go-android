abstract fun getItemsCursor(    selection: String,     selectionArgs: Array<String>,     sort: String): Cursor
Content copied to clipboard

Get a cursor containing items from the playlist. The items selected and sort order will be based upon the provided parameters and may not match the order in which the SDK will process them.

Return

The cursor of items for the playlist

Parameters

selection

A filter declaring which rows to return, formatted as an SQL WHERE clause (excluding the WHERE itself). Passing null will return all rows.

selectionArgs

You may include ?s in selection, which will be replaced by the values from selectionArgs, in the order that they appear in the selection. The values will be bound as Strings.

sort

How to order the rows, formatted as an SQL ORDER BY clause (excluding the ORDER BY itself). Passing null will use the default sort order, which will match the order in which the SDK processes the playlist.
