This is an SDK internal class which Dokka does not provide a way to hide from the public SDK. This class provides the Android Service for the main download service. It has historically been stored within the service package alongside ServiceStarter classes which are required for client integration. Dokka does not provide a method to hide individual Java classes within a package, and therefore this appears in the public documentation. However it is not a public class and the internal implementation should not be relied upon by the client application.
