HLSAssetBuilder

open class HLSAssetBuilder : AssetBuilder<T>

The HLSAssetBuilder is used to build the parameters for an HLS asset download. The build method will then produce an HLSAssetParams which can be passed to createHLSSegmentedAsset or createHLSSegmentedAssetAsync to create a new asset.
