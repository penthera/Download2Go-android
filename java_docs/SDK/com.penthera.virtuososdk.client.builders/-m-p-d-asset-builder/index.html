MPDAssetBuilder

open class MPDAssetBuilder : AssetBuilder<T>

The MPDAssetBuilder is used to build the parameters for an MPD asset download. The build method will then produce an MPDAssetParams which can be passed to createMPDSegmentedAsset or createMPDSegmentedAssetAsync to create a new asset.
