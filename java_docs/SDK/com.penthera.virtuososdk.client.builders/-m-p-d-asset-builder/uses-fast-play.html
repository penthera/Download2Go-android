usesFastPlay

open fun usesFastPlay(useFastPlay: Boolean): MPDAssetBuilder

Set if this asset should support fastplay. If set then this asset will be setup for fastplay regardless of if it is queued for download. Fastplay downloads do not report progress using the same observers as queued assets, instead they download silently. The queue observers will be notified of an update upon completion and the asset will be marked as fastplay ready, which can be checked with fastPlayReady.

Return

The asset builder for chaining.

Parameters

useFastPlay

true to use fastplay for the asset. Default is false.
