pauseDownload

abstract fun pauseDownload(aAsset: IAsset)
Content copied to clipboard

Pause an asset from being downloaded. If the asset is already queued then it will remain in the queue but will not be downloaded. It will not block other items in the queue from downloading. If the download has started then the item will stop downloading but any content already stored will be retained and it will count towards any download limits. If the download is already complete this method will have no effect.
