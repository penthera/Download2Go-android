createSegmentedAsset

abstract fun createSegmentedAsset(aAssetId: String, aMetadata: String): ISegmentedAsset

Download a segmented video, where you need to manually add segments to the asset later. This method leaves the newly-created asset instance in an un-queued state, so you can then add segments to it (via ISegmentedAsset.addSegments). Once you've added all segments, you should add the asset to the download queue.

Return

Concrete implementation of the ISegmentedAsset interface, or null if an error occurred.

Parameters

aAssetId

Caller-provided ID that Virtuoso should assign to the asset, will be considered as unique for business logic rules. See IAssetManager summary for more details.

aMetadata

Metadata that Virtuoso should associate with the asset. Typically a JSON string. Can be null.
