abstract fun createFileAsset(aAssetUrl: String, aAssetId: String, aMimeType: String, aMetaData: String): IFile

Download a single (non-segmented) file.

Return

Concrete instance of an IFile implementor, or null if an error occurred.

Parameters

aAssetUrl

Remote URL where the asset (a single file) should be downloaded from

aAssetId

Caller-provided ID that Virtuoso should assign to the asset, will be considered as unique for business logic rules. See IAssetManager summary for more details.

aMimeType

Expected mime type. If non-null, Virtuoso will use this value verify the validity of the file mime type during download.

aMetaData

Metadata that Virtuoso should associate with the asset. Typically a JSON string. Can be null.
