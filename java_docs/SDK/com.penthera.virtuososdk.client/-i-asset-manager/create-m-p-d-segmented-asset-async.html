Parameters

aObserver

Observer to be notified when the asset has been generated.

aManifestUrl

Remote URL of the manifest

aDesiredVideoBitRate

Virtuoso uses the closest bitrate, which does not exceed this one, when generating from a manifest. A value of 1 means "use the lowest bitrate." If there's no profile of lower bitrate than aDesiredBitRate, Virtuoso will select the lowest bitrate profile. A value of Integer.MAX_VALUE means "use the highest bitrate."

aDesiredAudioBitRate

Virtuoso uses the closest audio bitrate, which does not exceed this one, when generating from a manifest. If all availalbe bitrates exceed this value then the lowest bit rate is chosen. The value must be between 1 and Integer.MAX_VALUE.

aAssetId

Caller-provided ID that Virtuoso should assign to the asset, will be considered as unique for business logic rules. See IAssetManager summary for more details.

aMetadata

Metadata that Virtuoso should associate with the asset. Typically a JSON string. Can be null.

aAddToQueue

If true, Virtuoso will automatically add the asset to the download queue.

aPermObserver

The observer to be notified of any permission checks when added.
