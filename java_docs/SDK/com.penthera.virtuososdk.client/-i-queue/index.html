IQueue

interface IQueue : IAssetProvider
Content copied to clipboard

Provides access to the Assets that are queued for downloads. The queue's IAssetProvider implementation provides an ordered list of assets queued for download. The getCursor implementations will also default to sorting on the queues position. An instance of the queue can be retrieved through the IAssetManager of Virtuoso.
