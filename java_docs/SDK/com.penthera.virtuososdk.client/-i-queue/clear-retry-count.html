clearRetryCount

abstract fun clearRetryCount(aId: Int): Boolean

Clear the retry count on an asset so that the SDK will attempt further retries on download. The max retry condition is met when an asset's retry count reaches the max retry limit getDownloadRetryLimit. Once the SDK reaches the retry limit it will no longer attempt to download the asset until this method is called. This will only clear the retry count if the asset has reached the limit.

Return

True if the max retry condition was removed

Parameters

aId

The id of the Asset whose retry state should be cleared
