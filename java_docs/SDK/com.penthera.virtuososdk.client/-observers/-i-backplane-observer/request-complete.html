requestComplete

abstract fun requestComplete(    callbackType: Int,     result: Int,     errorMessage: String)
Content copied to clipboard

Virtuoso has completed its interaction with the Backplane. The specific action is identified by the callbackType parameter, which will be one of the values in Common.BackplaneCallbackType. Any additional error or success codes will be identified by the result parameter, which will be one of the values in Common.BackplaneResult.
