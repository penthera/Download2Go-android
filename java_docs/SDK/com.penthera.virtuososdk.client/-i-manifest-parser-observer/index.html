IManifestParserObserver

interface IManifestParserObserver

This interface provides a configurable mechanism for altering the output of the manifest parser before the Download 2 Go SDK begins downloading an asset. It provides methods to alter the URLs of each segment within the manifest, and update the asset properties prior to the asset being added to the queue. The class implementing IManifestParserObserver is instantiated within the service process of the SDK and is called directly within the parsing process. It may be called when the app is in the background as the result of manifest parsing continuing after the app is backgrounded. It will only be called when the SDK determines that the device is online. The IManifestParserObserver is provided to the engine through the IBackgroundProcessingManager interface, which itself is registered via the Android manifest.
