abstract fun willAddToQueue(    segmentedAsset: ISegmentedAsset,     assetManager: IAssetManager,     context: Context)
Content copied to clipboard

The SDK has finished parsing the asset and is about to add the asset to the download queue. This method gives the enclosing application a chance to modify the asset's configuration options (E.G. start window, end window, asset download limit) before the asset is added to the download queue and downloading begins.

Parameters

segmentedAsset

Generated segmented asset.

assetManager

An asset manager instance which can be used for saving any changes

context

A context that can be useful when fetching resources to process the add to queue action
