shouldUpdateForegroundServiceNotificationOnIntent

abstract fun shouldUpdateForegroundServiceNotificationOnIntent(context: Context, reasonIntent: Intent): Boolean
Content copied to clipboard

Called to check if the provider wishes to generate a new notification for an intent. If this method returns true then the getForegroundServiceNotification method will be called with the intent.

Return

true if a new notification will be generated for this intent.

Parameters

context

The context used by the calling service for notification related calls.

reasonIntent

The intent for which a notification may be generated.
