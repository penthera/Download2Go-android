Manually request a sync This method has been deprecated. It is not necessary to manually request sync as the service will sync the device at the necessary intervals to ensure it is up to date with the server. Virtuoso's background service will attempt to sync with the Backplane at least once every 24 hours, but will also sync upon a number of internal events. The sync is internally rate limited to ensure it cannot be called too frequently. Completion of a sync is reported through the requestComplete callback in IBackplaneObserver with a callback type of BackplaneCallbackType.SYNC.
