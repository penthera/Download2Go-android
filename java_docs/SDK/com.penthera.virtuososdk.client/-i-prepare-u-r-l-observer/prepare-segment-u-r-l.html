prepareSegmentURL

abstract fun prepareSegmentURL(segmentUrl: String, segment: ISegment, asset: ISegmentedAsset): String

This method is called directly prior to beginning download of each segment within a segmented asset. It must return either an updated URL to use for the download, or null to continue with the existing url. This class should only be necessary if segment URLs need to be updated dynamically at the point of download. If the urls can be updated at the point when the asset is added and the manifest parsed then it is more efficient to use the IManifestParserObserver. If this class is returned from the background processing manager then this method will be called for every segment download attempt, so it should be as efficient as possible in returning updated URLs. Note: the segment details provided to this method makes it possible to check if the segment download is being attempted for the first time or has failed a download previously.

Return

An updated download URL, or null to continue using the original URL provided.

Parameters

segmentUrl

The original segment URL

segment

The segment corresponding to the URL to be downloaded, contains information about previous download attempts

asset

The asset to which the segment belongs.
