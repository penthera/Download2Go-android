IPrepareURLObserver

interface IPrepareURLObserver

This interface provides a configurable mechanism to allow modification of the request URLs for segments or keys (HLS) prior to requests being made. The class implementing IPrepareURLObserver is instantiated within the service process of the SDK and is called directly within the downloading process. It may be called when the app is in the background as downloading continues while the app is backgrounded. It will only be called when the SDK determines that the device is online. The IPrepareURLObserver is provided to the engine through the IBackgroundProcessingManager interface, which itself is registered via the Android manifest.
