interface IFile : IAsset
Content copied to clipboard

The interface to flat files in Virtuoso. Virtuoso can apply several checks to validate a downloaded file:

  1. The observed mime-type matches the expected mime-type.
  2. The observed file size matches the expected file size
  3. The observed file size matches the the content-length advertised by the remote HTTP server
We strongly encourage you to specify these 'expected' values. If you don't, Virtuoso has no way to guarantee the validity of the downloaded file. You can set the expected values when you create a concrete instance of an IFile implementor with createFileAsset.
