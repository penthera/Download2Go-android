complete

abstract fun complete(    aSegmentedAsset: ISegmentedAsset,     aError: Int,     addedToQueue: Boolean)
Content copied to clipboard

This method is called to indicate that the creation of an ISegmentedAsset from a remote manifest has been completed. It provides the final outcome of the manifest parsing, which can be either success or an error reason, and also the segmented asset providing that the parser did not reach an error before the asset could be created. This method gives the enclosing application a chance to modify the asset's configuration options (E.G. start window, end window, asset download limit) after the asset is added to the download queue and while downloading begins.
