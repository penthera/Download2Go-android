IHLSManifestRenditionSelector

interface IHLSManifestRenditionSelector

Virtuoso will automatically select video renditions based on the supplied filters and maximum bitrate provided during asset creation. If you have more advanced requirements for rendition selection of an HLS asset that cannot be fulfilled by the automatic selection process, you may define your own selection by implementing a IHLSManifestRenditionSelector and registering it with the background processing manager. Prior to selecting the rendition for download, the SDK will provide the IHLSManifestRenditionSelector a list of IHLSVideoRendition objects. You may choose which one you would like to use for download, and return that to the SDK.
