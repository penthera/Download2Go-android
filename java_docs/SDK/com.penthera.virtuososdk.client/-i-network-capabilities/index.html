interface INetworkCapabilities

The NetworkCapabilities class provides a method to check the available transport being used by the SDK. The class methods and constants are named inline with the similar terms in the Android class android.net.NetworkCapabilities, which was made available in API level 21. The SDK supports older Android versions and therefore cannot directly use those classes.
