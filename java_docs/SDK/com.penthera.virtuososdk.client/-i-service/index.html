interface IService

Some calls within Virtuoso require direct communication with the background download service. These calls consist of pausing / resuming downloads, ascertaining current status of the service and retrieving network throughput data. This interface provides the means for clients to create a persistent connection only when they need it. Use this interface to:

  • Pause Downloads.
  • Resume Downloads.
  • Retrieve the current status.
  • Determine the real-time download performance during the last 15 seconds.
  • Determine the download performance average for the currently downloading asset.
  • Determine the download performance average for the overall engine.
This interface can be accessed through a call to getService of Virtuoso.
