setHeadroom

abstract fun setHeadroom(aHeadroom: Long): ISettings
Content copied to clipboard

Change the setting for headroom. Headroom is the minimum amount of free space that Virtuoso must preserve on disk. For instance, if headroom is 1GB and the device has 1.2GB free disk space, Virtuoso would download only 200MB of a large asset and then block the download on storage in order to maintain the headroom. The default value is 1 GB. The minimum headroom permitted is 500MB. The new setting is applied automatically after being set. If the new value differs from the current setting, then a callback is made to settingChanged callback in IEngineObserver. Applications can listen for changes to the configuration value by implementing the settingChanged. Flags reported through that API are: SETTING_HEADROOM

Return

The current settings object, to allow for call chaining.
