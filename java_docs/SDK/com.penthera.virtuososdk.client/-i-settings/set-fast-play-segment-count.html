setFastPlaySegmentCount

abstract fun setFastPlaySegmentCount(count: Int): ISettings
Content copied to clipboard

Sets the count for the number of segments to download for Fast Play assets. The new setting is applied automatically after being set. Applications can listen for changes to the configuration value by implementing the settingChanged method in IEngineObserver. Flags reported through that API are: SETTING_FASTPLAY_COUNT

Return

The current settings object, to allow for call chaining.

Parameters

count

The number of segments to download for Fast Play assets.
