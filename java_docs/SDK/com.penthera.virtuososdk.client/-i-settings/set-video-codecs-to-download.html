setVideoCodecsToDownload

abstract fun setVideoCodecsToDownload(codecs: Array<String>): ISettings

Sets the video codec formats allowed for content downloads The new setting is applied automatically after being set. Applications can listen for changes to the configuration value by implementing the settingChanged method in IEngineObserver. Flags reported through that API are: SETTING_CODECS_TO_DOWNLOAD

Return

The current settings object, to allow for call chaining.

Parameters

codecs

List of desired video codec formats
