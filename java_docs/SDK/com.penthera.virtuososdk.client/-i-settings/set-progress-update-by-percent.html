setProgressUpdateByPercent

abstract fun setProgressUpdateByPercent(aPercent: Int): ISettings

Configure the frequency of progress updates during a download.

While downloading, Virtuoso will report back updates through the Content Uri, the IQueueObserver or via Broadcasts. This method allows a caller to configure the frequency of those updates by the percent of asset downloaded. The setting value should range from 0 to 100. A value of 0 or less resets the progress update for percentage to the default of 1%. A value of 100 turns progress updates off at a percentage level. The new setting is applied automatically after being set. Applications can listen for changes to these configuration values by implementing the settingChanged method in IEngineObserver. Flags reported through that API are: SETTING_PROGRESS_UPDATE_PERCENT

Return

The current settings object, to allow for call chaining.

Parameters

aPercent

The new value to use for configuring the update frequency.
