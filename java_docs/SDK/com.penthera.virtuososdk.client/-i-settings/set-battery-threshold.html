setBatteryThreshold

abstract fun setBatteryThreshold(aThreshold: Float): ISettings
Content copied to clipboard

Change the setting for battery threshold. The battery threshold is applied to prevent downloading when the battery level is too low. This feature is only available on Android and not iOS because Android has the capability to monitor battery level at all times whereas iOS cannot monitor when in the background. The battery threshold does not apply if the device is charging and the current charge is greater than 10%. The battery threshold may not be honored on a simulator. Acceptable values are [0,1], where 0 means 'always download' and 1 means 'download only while charging'. Virtuoso will download only if the battery charge exceeds this value. The default value is 0.5 (50%). The new setting is applied automatically after being set. If the new value differs from the current setting, then a callback is made to settingChanged callback in IEngineObserver Applications can listen for changes to the configuration value by implementing the settingChanged. Flags reported through that API are: SETTING_BATTERY_THRESHOLD

Return

The current settings object, to allow for call chaining.
