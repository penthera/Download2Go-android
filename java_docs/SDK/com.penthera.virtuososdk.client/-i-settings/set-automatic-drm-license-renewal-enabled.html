abstract fun setAutomaticDrmLicenseRenewalEnabled(enabled: Boolean): ISettings

Sets the enabled state of automatic drm license renewals The new setting is applied automatically after being set. Applications can listen for changes to the configuration value by implementing the settingChanged method in IEngineObserver. Flags reported through that API are: SETTING_DRM_LICENSE_RENEWAL_ENABLED

Return

The current settings object, to allow for call chaining.
