IAssetProvider

interface IAssetProvider

Provides access to Asset data through a Cursor. The Cursor uses the Content Uri for querying the Asset database. Where the cursor returned in these methods contains a local playback URI for a segmented asset manifest under com.penthera.virtuososdk.client.database.AssetColumns#PLAYLIST , it will only include the URI path and not the scheme or authority. These can be found from com.penthera.virtuososdk.client.Virtuoso#getHTTPProxyBaseUri()
