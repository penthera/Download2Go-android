LicenseManager

open class LicenseManager : ILicenseManager

Default implementation of the ILicenseManager interface. This class should be extended to override the getLicenseAcquistionUrlForAsset method. For the SDK to pick up the new LicenseManager implementation class an application must replace the meta-data tag named "com.penthera.virtuososdk.license.manager.impl" and provide the full class name as the value. 


<meta-data tools:replace="android:value"
         android:name="com.penthera.virtuososdk.license.manager.impl"
         android:value="com.penthera.harness.drm.HarnessLicenseManager"/>
