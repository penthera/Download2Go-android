open fun buildWithAsset(context: Context, asset: IAsset): ILicenseManager
Content copied to clipboard

Retrieve the LicenseManager instance with an intialised IAsset. The returned instance is of the class specified in the meta-data tag named com.penthera.virtuososdk.license.manager.impl in the applications manifest. If the application has not replaced the SDK meta-data then an instance of LicenseManager is returned. The default implementation of this method returns a chained call of getInstance(Context) and withAsset(IAsset).

Return

Implementation of an ILicenseManager with an initiliased asset.

Parameters

context

Context to use for reading the manifest and retrieving the class name.

asset

The IAsset to be used by the LicenseManager.
