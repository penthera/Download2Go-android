executeKeyRequest

open fun executeKeyRequest(uuid: UUID, request: MediaDrm.KeyRequest): Array<Byte>

The executeKeyRequest method can be derived from ILicenseManager and is reponsible for returning the pure widevine DRM license details as a byte array. These details are put directly into the android MediaDrm framework using provideKeyResponse . The android framework does not handle any of the metadata returned in the castlabs response.

Return

The byte array, suitable for passing into MediaDrm#provideKeyResponse
