CastLabsLicenseManager

abstract class CastLabsLicenseManager : LicenseManager
Content copied to clipboard

An extension of the default LicenseManager which provides utility methods for working with the castlabs request and response formats. This class must be extended in order for the application to provide the necessary parameters to build URIs for DRM requests. The derived class must be registered with the SDK via the Android manifest, as documented in LicenseManager.
