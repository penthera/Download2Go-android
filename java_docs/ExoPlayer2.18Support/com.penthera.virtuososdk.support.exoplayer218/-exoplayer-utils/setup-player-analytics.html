setupPlayerAnalytics

open fun setupPlayerAnalytics(@NonNull context: Context, @NonNull player: Player, @Nullable assetAccess: IAssetAccess, playingUrl: String)

Intialise playback analytics for a player that has not been created using a setupPlayer() method from this utility class.

If the player is being manually constructed to play media within an application containing a Penthera SDK for either Download or PlayAssure, then this method will configure player analytics collection for the SDK. It is intended for use with Streaming or Play Assure assets. If a player is manually constructed for download assets then please use the overloaded method and specify the type.

Parameters

context

An android context to associate with the player session.

player

The player for which to register analytics listeners.

assetAccess

IAssetAccess instance used to find assets in the download SDK, may be null if only used for playassure or streaming

playingUrl

The url that is to be played and should be associated with the analytics.
