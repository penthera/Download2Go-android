The HLS Media Source Factory provides an option to allow Chunkless preparation, where streams are prepared without downloading initial chunks. This option enables the value to be set for the Hls factory when using the ExoplayerUtils methods.

Return

The value to be set on construction of the HlsMediaSource.Factory
