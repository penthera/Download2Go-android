VirtuosoFrameworkMediaDrm

@RequiresApi(value = 18)
class VirtuosoFrameworkMediaDrm : ExoMediaDrm

A direct copy of Exoplayer's FrameworkMediaDrm which exposes the underlying platform MediaDrm so that it can be shared. The exoplayer implementation is marked final and does not expose what the SDK needs.
