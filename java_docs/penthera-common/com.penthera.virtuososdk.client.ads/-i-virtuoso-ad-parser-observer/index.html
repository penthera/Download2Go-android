IVirtuosoAdParserObserver

interface IVirtuosoAdParserObserver
Content copied to clipboard

The Ad Parser Observer provides an interface which can be provided with the IClientSideAdsParserProvider to be informed of errors during parsing of Advertising definitions which are associated with assets. An advertising configuration failure will not prevent the associated asset from being downloaded or reported as successful. It is independent to the asset and will only be reported via this interface.
