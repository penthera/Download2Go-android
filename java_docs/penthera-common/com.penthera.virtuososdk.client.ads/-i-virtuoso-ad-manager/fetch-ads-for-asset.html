fetchAdsForAsset

abstract fun fetchAdsForAsset(aAsset: IAsset): IVideoAdPackage
Content copied to clipboard

If available, fetch the advertising package for an asset. The IVideoAdPackage contains methods to enable fetching the details of each available advert and the playback of the advert(s) through a VAST/VMAP definition.

Return

The advertising package containing details of available adverts, or null if the asset does not support adverts or none are currently available.

Parameters

aAsset

The asset for which advertising details are required.
