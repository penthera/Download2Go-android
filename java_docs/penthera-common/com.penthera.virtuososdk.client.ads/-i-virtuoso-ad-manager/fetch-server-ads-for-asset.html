fetchServerAdsForAsset

abstract fun fetchServerAdsForAsset(aAsset: IAsset): IServerDAIPackage
Content copied to clipboard

If available, fetch the Dynamic Ad Insertion package for an asset. The IServerDAIPackage contains methods to get the list of cue points for ads contained within the downloaded stream, it also contains a method to report media verification ids back to the server which are contained within the media metadata.

Return

The Dynamic Ad Insertion package containing details of adverts within the asset stream, or null if the asset does not support DAI.

Parameters

aAsset

The asset for which advertising details are required.
