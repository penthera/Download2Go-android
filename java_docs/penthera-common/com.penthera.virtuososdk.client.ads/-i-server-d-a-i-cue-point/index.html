IServerDAICuePoint

interface IServerDAICuePoint

This represents a cue point within the playback stream which will contain one or more ads. The cue point is simply a time period containing ads, which can be marked as played to help with controlling seeking and preventing skipping ad segments. Played flags do not persist across instances of the containing IServerDAIPackage, they are intended for a single playback only.
