interface IServerDAIPackage
Content copied to clipboard

The ServerDAIPackage represents the sequence of adverts encapsulated within an asset using Dynamic Ad Injection by the server. The ads are automatically played as part of the stream, this interface provides some details that may be required to manage playback and report impressions. IServerDAIPackage provides details on the media duration and ad insertion cue points that may be required to provide playback progress indicators or control seeking over the adverts. It also provides a method to report media verification ids for impression tracking to the server.
