getLastPermissionResponse

abstract fun getLastPermissionResponse(): IAssetPermission

Permission response from Backplane.

The permissions are requested from the backplane if the Max Account Downloads, Max Asset Downloads, or Max Asset Copies settings have been configured. They are also requested if Always Check Permissions setting is configured.

The permission is evaluated when added to the queue. If configured and the permission check is not required at time of queuing then the permission code in the the IAssetPermission permission code will be set to PERMISSION_AWAITING_PERMISSI0N until the permission is checked at time of download.

Return

The IAssetPermission or null if not yet added to queue.
