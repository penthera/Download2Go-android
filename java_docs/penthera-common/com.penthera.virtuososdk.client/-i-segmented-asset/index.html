ISegmentedAsset

interface ISegmentedAsset : IAsset

Interface to segmented assets in Virtuoso. A segmented asset consists of one or more segments. Virtuoso treats this collection of segments as an atomic unit when downloading. Developers can retrieve an instance by calling createSegmentedAsset() or one of its sibling methods in IAssetManager
