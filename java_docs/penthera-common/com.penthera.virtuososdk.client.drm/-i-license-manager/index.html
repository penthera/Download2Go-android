ILicenseManager

interface ILicenseManager

Interface to the License Manager. The License Manager is used to retrieve and store licenses so that Assets can be played back whilst offline. A default implementation is provided by LicenseManager. The LicenseManager class should be extended to override the getLicenseAcquistionUrlForAsset method.
