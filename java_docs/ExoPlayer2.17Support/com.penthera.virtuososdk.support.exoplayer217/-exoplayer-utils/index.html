ExoplayerUtils

open class ExoplayerUtils

Utility class that provides functionality to more easily link the Virtuoso SDK to Exoplayer version 2.15.* This class provides the ability to fully initialize a ExoPlayer instance for playback or to create the various components needed, (DrmSessionManager, MediaSource), to easily integrate playback of offline content through the Virtuoso SDK.
