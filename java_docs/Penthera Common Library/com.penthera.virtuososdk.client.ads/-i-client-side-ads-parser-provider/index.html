IClientSideAdsParserProvider

interface IClientSideAdsParserProvider

This interface enables the application to provide ad server URLs and observers for error reporting to the SDK parser service. The methods in this class will each run within the service process, and will be called to fetch an updated list each time an observer or url resolver is required. The class implementing IClientSideAdsParserProvider is instantiated within the service process of the SDK and is called directly within the parsing process. It may be called when the app is in the background as the result of manifest parsing continuing after the app is backgrounded. It will only be called when the SDK determines that the device is online. The IClientSideAdsParserProvider is provided to the engine through the IBackgroundProcessingManager interface, which itself is registered via the Android manifest.
