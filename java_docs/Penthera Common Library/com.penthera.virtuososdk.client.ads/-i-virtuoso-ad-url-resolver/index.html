IVirtuosoAdUrlResolver

interface IVirtuosoAdUrlResolver

IVirtuosoAdResolver can be supplied to the IClientSideAdsParserProvider to allow custom ad content to be used for specific assets Add an instance of IVirtuosoAdUrlResolver to the list provided by IClientSideAdsParserProvider prior to parsing any asset manifests that have related ad content Multiple IVirtuosoAdUrlResolvers can be provided via the IClientSideAdsParserProvider. The SDK will check the added IVirtuosoAdUrlResolvers in the order that they are listed and will return the first URL that it finds
