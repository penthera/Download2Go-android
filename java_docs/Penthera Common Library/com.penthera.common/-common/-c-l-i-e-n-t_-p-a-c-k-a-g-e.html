CLIENT_PACKAGE

val CLIENT_PACKAGE: String = "com.penthera.virtuososdk.client.pckg"

Name of the XML tag you must provide in the Android manifest. This string identifies the client and should be used as the client's provider authority. This string is also prepended to the notification Intent action strings, so that clients can intercept updates from the service and update the notification UI. Example snippet from your Android manifest: 


<meta-data android:name="com.penthera.virtuososdk.client.pckg" android:value="com.penthera.reference.provider" />
