abstract fun setStartWindow(start: Long)

Set the start window for playback.

Attempts to access or play this asset through a Virtuoso API will return null until this time has passed.

A newly-created asset defaults to having a start window of 'now'.

Parameters

start

A timestamp, represented as the number of seconds since the unix epoch.
