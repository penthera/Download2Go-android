ExoplayerDrmSessionManager

open class ExoplayerDrmSessionManager : DrmSessionManager

Provides a DRM session manager which can load asset details from the Penthera SDK. If the playing media item comes from a downloaded asset then the session manager will try to create a session based upon offline keys. It will refresh expired keys if possible.
