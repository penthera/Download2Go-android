Penthera Common Library

The Penthera common library provides supporting functions to the Download and PlayAssure SDKs
for server communications, event logging, and debugging.

It contains all of the constants and enumerations which are used by both public SDKs. The common library is a
gradle dependency of each public SDK module, so will be included into your build automatically.
