SupportDrmSession

open class SupportDrmSession : IVirtuosoDrmSession<T> , MediaDrm.OnEventListener

An implementation of the IVirtuosoDrmSession interface for the default MediaCrypto class which is used by the DRM framework. This is used by the SupportDrmSessionManager. If you are managing DRM sessions with a custom manager then it is unlikely that this will be of benefit to the implementation.
