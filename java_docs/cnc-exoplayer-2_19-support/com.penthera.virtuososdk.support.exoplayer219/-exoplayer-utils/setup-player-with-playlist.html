setupPlayerWithPlaylist

open fun setupPlayerWithPlaylist(@NonNull playerView: StyledPlayerView, @NonNull assets: List<String>, @NonNull assetAccess: IAssetAccess, @Nullable options: ExoplayerUtils.PlayerConfigOptions, @Nullable playbackConfig: ExoplayerUtils.PlaylistPlaybackConfig): Player

Create and initialize a ExoPlayer instance for playback of the provided assets as a playlist using the provided options This method will create a ExoPlayer instance, link it to the provided StyledPlayerView and initialize it for playback with a MediaSource that points to the passed in IAsset. The returned ExoPlayer instance will be fully prepared for playback. If an error is encountered or the IAsset is not ready for playback, this method will return null.

Return

A fully initialized ExoPlayer that is ready to play back the specified IAsset items in a playlist.

Parameters

playerView

the StyledPlayerView used to create a ExoPlayer.

assets

List of string asset UUIDs that make up the playlist assets

assetAccess

a IAssetAccess used for accessing asset information.

options

PlayerConfigOptions used to configure the player, media source and drm for playback.

playbackConfig

used to configure the playback type used for playlist items. May be null in which case, the standard playback url will be used for each item.

Throws

MalformedURLException

when one of the the IAsset playback url is not valid.
