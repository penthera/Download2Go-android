processingMode

abstract val processingMode: PlayAssureProcessingMode

PlayAssure player interaction (throttling) with cache handling

Basic - No player interaction. Cache downloads are at the current player bitrate unless lowestBitrateEnabled is set.

Monitor - Player is throttled with decreased bandwidth based on the current player bitrate. Cache downloads are at the current player bitrate.

Backstop - Player is throttled with decreased bandwidth based on a fixed target player bitrate. Cache downloads are at a fixed target player bitrate.
