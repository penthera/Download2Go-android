PlayAssure SDK Library

This SDK provides the PlayAssure features, which may be used independently of the Download SDK or within the same application. To use the PlayAssure SDK, it must be added in your project gradle dependencies. It will automatically include the penthera common library.

PlayAssure SDK Library

This package contains all the public interfaces required to use the PlayAssure SDK on Android. Simply instantiate the PlayAssureManager class with a configuration and a source media URL, wait for it to initialize and then pass the streamingURL from the manager to your player.
