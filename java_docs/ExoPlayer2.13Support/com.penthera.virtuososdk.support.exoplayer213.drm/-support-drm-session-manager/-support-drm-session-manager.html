constructor(context: Context, uuid: UUID, asset: IAsset, optionalKeyRequestParameters: HashMap<String, String>, playbackLooper: Looper, eventHandler: Handler, eventListener: SupportDrmSessionManager.EventListener)

Create a DrmSessionManager to be used during media playback.

Parameters

context

The context to be used by the manager.

uuid

The UUID of the drm scheme currently only supports widevine WIDEVINE_UUID.

asset

The IAsset that any license requests will be made for.

optionalKeyRequestParameters

An optional map of parameters to pass as the last argument to getKeyRequest. May be null.

playbackLooper

The looper associated with the media playback thread. When using ExoPlayerV2 this can be null.

eventHandler

A handler to use when delivering events to eventListener. May be null if delivery of events is not required

eventListener

A listener of events. May be null if delivery of events is not required.

Throws

UnsupportedDrmException

exception thrown if DRM type in unsupported by device
