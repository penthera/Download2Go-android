resumeDownloads

open fun resumeDownloads(): Boolean

Requests that the engine is resumed from pause to continue downloads. Reports success of the request. The can fail if the current state is already RESUMING or RESUMED, or if the service cannot be accessed.

Return

Returns true if the request was successfully passed to the service, false otherwise.
