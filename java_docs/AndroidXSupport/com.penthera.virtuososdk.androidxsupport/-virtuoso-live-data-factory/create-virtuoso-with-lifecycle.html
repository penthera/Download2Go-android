createVirtuosoWithLifecycle

@MainThread
open fun createVirtuosoWithLifecycle(@NonNull lifecycleOwner: LifecycleOwner, @NonNull context: Context): Virtuoso

Create the Virtuoso SDK interface to be used by the factory and associate it with a LifecycleOwner. Use this method if the factory will be the main interface to the SDK, the underlying Virtuoso object is made available for other direct interactions such as creating new assets or accessing settings. The lifecycle of the LifecycleOwner will be used for all data delivered by the factory. Note that this method should be run on the main application thread.

Return

The Virtuoso top level SDK interface, made available for other interactions than those using LiveData.

Parameters

lifecycleOwner

The lifecycle owner that will control the lifecycle of all objects created with the factory.

context

The context used to create the Virtuoso SDK instance.
