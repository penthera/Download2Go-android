createLifecycleWithVirtuoso

@MainThread
open fun createLifecycleWithVirtuoso(@NonNull lifecycleOwner: LifecycleOwner, @NonNull virtuoso: Virtuoso)

Associate the factory with an existing instance of the Virtuoso SDK and with a LifecycleOwner. Use this method if the Virtuoso SDK has already been created for other access and the factory is used for some additional views on the asset data. The lifecycle of the LifecycleOwner will be used for all data delivered by the factory. Note that this method should be run on the main application thread.

Parameters

lifecycleOwner

The lifecycle owner that will control the lifecycle of all objects created with the factory.

virtuoso

The Virtuoso SDK interface to be used by the factory.
