Download2Go AndroidX Extension Support Library

The core package in this support library, providing LiveData classes for each of the main SDK concepts. The VirtuosoLiveDataFactory is the first point of entry for this extension SDK. Once this class has been instantiated with a LifeCycleOwner then it can be used to fetch LiveData classes which will be managed with the associated lifecycle.
