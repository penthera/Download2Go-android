PlayAssure SDK Library/com.penthera.playassure

Package-level declarations

This package contains all the public interfaces required to use the PlayAssure SDK on Android. Simply instantiate the PlayAssureManager class with a configuration and a source media URL, wait for it to initialize and then pass the streamingURL from the manager to your player.

PlayAssureEngineState

enum PlayAssureEngineState : Enum<PlayAssureEngineState>

The startup state of the SDK. When instantiated, the SDK will check setup device storage for temporary files and will check for existing licensing and registration details. If these are unavailable then the SDK will be paused in the Unregistered state until the SDK is successfully initialized. This initialization is a one time operation and the SDK will retain the state between sessions. Once started, the SDK will process the requested asset and will be ready to begin playback.
