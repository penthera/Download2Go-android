Create and initialize a Player instance for playback of the specified asset using the provided options This method will create a Player instance, link it to the provided PlayerView and initialize it for playback with a MediaSource that points to the passed in IAsset. The returned Player instance will be fully prepared for playback. If an error is encountered or the IAsset is not ready for playback, this method will return null.

Return

A fully initialized Player that is ready to play back the specified IAsset.

Parameters

playerView

the PlayerView used to create a Player.

assetManager

a IAssetAccess used for accessing asset information .

asset

the IAsset to initialize for playback.

useFastPlay

boolean value true indicates that the specified IAsset should be initialized for FastPlay playback.

options

PlayerConfigOptions used to configure the player, media source and drm for playback.

Throws

java.net.MalformedURLException

when the IAsset playback url is not valid.
