A custom implementation of the exoplayer DrmSessionManagerProvider interface that delivers a DRMSessionManager which is aware of offline DRM licenses managed by the SDK. This provider is used within the ExoplayerUtils methods, which provide the easiest client integration to use exoplayer with SDK managed offline DRM. However it can be manually constructed and provided to a custom DRM implementation with Exoplayer if required.
